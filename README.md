# Projeto de provas de teste funcional

Projeto Maven configurado para o IntelliJ com a seguinte estrutura:

- `specs/`: enunciados em Markdown
- `src/`: implementações por prova
- `test/`: pastas separadas por prova para os testes

## Como abrir no IntelliJ

1. Abra a pasta do projeto no IntelliJ.
2. Importe o `pom.xml` como projeto Maven.
3. O IntelliJ reconhecerá `src` como código-fonte e `test` como código de teste.

As implementações foram criadas para servir de base para os testes pedidos nos enunciados.
